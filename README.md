## Installing (Kali Linux):

```
apt-get update

git clone https://github.com/TechEthical/GBomber

cd GBomber

python GBomber.py
```


## Installing (Termux):

```
pkg update

pkg install git

pkg install python

git clone https://github.com/TechEthical/GBomber.git

cd GBomber

chmod +x GBomber.py

python GBomber.py
```
